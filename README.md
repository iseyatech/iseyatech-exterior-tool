# 伊勢屋テック 外装塗装 提案イメージ生成ツール

ブラウザーだけで動く公開用ツールです。  
社員は URL を開くだけで利用できます。

## できること
- 外観写真アップロード
- 部位別レイヤー作成（外壁 / 入り口 / 柱 / シャッター / 看板 / その他）
- ブラシで塗るように範囲指定
- 外壁素材アップロード
- 色番データアップロード（CSV / JSON）
- 色のみ / 素材のみ / 両方の反映
- JPG出力
- ローカル保存 / 読み込み
- 設定JSON出力

## 公開方法（Vercel）
1. このフォルダ一式を GitHub にアップロード
2. Vercel にログイン
3. `Add New` → `Project`
4. GitHub リポジトリを選択
5. Framework Preset は `Other`
6. Deploy を押す
7. 発行された URL を社員に共有

## ファイル構成
- `index.html` : 本体
- `vercel.json` : Vercel 用設定
- `README.md` : 説明書

## 色データの形式

### CSV例
```csv
code,name,hex
N-90,ホワイト,#F3F3EF
N-50,グレー,#8D8D87
N-20,チャコール,#474742
```

### JSON例
```json
[
  { "code": "N-90", "name": "ホワイト", "hex": "#F3F3EF" },
  { "code": "N-50", "name": "グレー", "hex": "#8D8D87" }
]
```

## 注意
この版はブラウザー完結のため、共通DBやログイン管理はありません。  
まずは現場導入用の公開版として使い、次段階で案件管理・素材共通マスター化・AI自動マスク化を追加する前提です。
