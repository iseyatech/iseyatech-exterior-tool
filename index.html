Iseyatech / Public Browser Tool

外装塗装 提案イメージ生成ツール

外観写真をアップロードし、部位ごとに塗るように範囲指定して、素材と色番を反映できます。ブラウザーだけで動作し、JPG出力・ローカル保存に対応しています。

案件情報

保存名や出力ファイル名に利用

① 外観写真

正面写真を1枚アップロード

② 部位とレイヤー

外壁・入り口など部位ごとに編集

③ 範囲指定

四角ではなく、塗るように指定
ブラシサイズ28 px
ブラシの柔らかさ0.6

④ 外壁素材

カタログ画像を登録

⑤ 色番データ

CSV / JSON を登録

⑥ 反映設定

色のみ・素材のみ・両方に対応
色の強さ55%
素材の強さ36%
仕上がり濃度100%

色データの形式

この形式ならそのまま読めます
CSV例
code,name,hex
N-90,ホワイト,#F3F3EF
N-50,グレー,#8D8D87
N-20,チャコール,#474742
JSON例
[
  { "code": "N-90", "name": "ホワイト", "hex": "#F3F3EF" },
  { "code": "N-50", "name": "グレー", "hex": "#8D8D87" }
]

プレビュー

手順：レイヤー追加 → 部位選択 → キャンバス上を塗る → 素材/色を反映 → JPG出力
まず建物の外観写真をアップロードしてください
URLを開いた社員がそのまま使える前提の公開版です。

公開方法

このフォルダをそのまま Vercel へ載せれば公開できます
  1. この一式を GitHub にアップロード
  2. Vercel でその GitHub リポジトリを Import
  3. Deploy を押す
  4. 発行URLを社員へ共有
